## 二、软件用途

Lattice-Planner 是一款面向个人的规划与任务管理软件，用于帮助用户在「执行」「学习」「规划」等不同思维模式下，更有条理地安排与回顾工作与生活。主要用途包括：

1. **任务管理**：创建、完成、搁置、删除任务；设置截止日期、精力需求、心理负担、期望时间段；支持按关键词与日期范围搜索与筛选；支持按时间段或精力分组的视图切换。  
2. **目标管理**：创建与管理中长期目标，将任务关联到目标，归档已达成目标，并通过事件驱动机制自动更新目标进度。  
3. **笔记管理**：添加与查看笔记，支持多种笔记类型，用于复盘与知识积累。  
4. **用户偏好与思维模式**：设置每屏任务数、是否显示未来任务、是否显示统计信息；在「执行」「学习」「规划」三种模式间切换，规划模式下可访问规划得分与 AI 总结。  
5. **规划得分与统计**：按日计算任务、目标、笔记三维度的规划得分（0–100），以折线图展示一段时间内的每日得分曲线。  
6. **AI 总结**：基于选定时间段的每日得分数据，自动生成自然语言总结与建议；支持接入外部大语言模型（如 Gemini）或使用内置本地规则总结。  
7. **桌面端与 DDL 提醒**：通过 Electron 桌面客户端（Lattice-Planner 客户端）访问 Web 服务，支持系统托盘、单实例运行；在用户登录状态下定期检查任务截止时间，对即将到期（1 天内、3 天内）或已过期的任务发送桌面通知提醒。

## 三、技术特点

1. **前后端分离与多端形态**：后端提供 RESTful 风格接口与页面渲染，前端通过浏览器或 Electron 嵌入页访问；桌面端与后端通过 Cookie 保持会话，实现登录状态同步与 DDL 检查。  
2. **核心与插件解耦的架构**：核心层仅包含任务、用户、笔记等实体与基础服务，通过 Spring 事件发布任务创建、完成、归档等事件；目标、洞察（得分与 AI 总结）等以插件形式通过事件监听扩展功能，核心层不依赖插件，便于维护与扩展。  
3. **多维度规划得分算法**：按任务截止日期归属统计，从任务（加权完成率与吞吐量因子）、目标（进度与推进覆盖面）、笔记（条数递减增益）三个维度计算每日 0–100 分，鼓励用户兼顾重要任务、长期目标与复盘记录。  
4. **AI 总结双通路**：支持配置外部 Gemini API 生成自然语言总结；在未配置或调用失败时自动回退到本地规则总结，保证功能可用性。  
5. **桌面端特性**：Electron 单实例锁、系统托盘、窗口隐藏到托盘、定时拉取登录状态与截止任务列表、按规则发送桌面通知（1 天/3 天/已过期），并具备连接失败时的本地错误页与重试能力。

## 四、运行环境

- **服务端**  
  - 操作系统：Windows / Linux / macOS 等支持 Java 运行环境的系统。  
  - 运行环境：JDK 17 及以上。  
  - 数据库：MySQL 5.7 或 8.x（需创建数据库并配置连接）。  
  - 可选：配置 Gemini API Key 以启用 AI 总结的在线模型调用。  

- **Web 端**  
  - 浏览器：Chrome、Edge、Firefox、Safari 等现代浏览器。  
  - 访问方式：通过服务端部署地址（如 http://localhost:8080）访问。  

- **桌面端（Lattice-Planner 客户端）**  
  - 操作系统：Windows 10/11（当前提供 Windows x64 构建）。  
  - 运行环境：Electron 运行时（随安装包提供）。  
  - 依赖：需先启动 Lattice-Planner 服务端，并保证客户端可访问服务端地址（默认 http://localhost:8080，可通过环境变量配置）。
